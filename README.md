# ⚔️ DQW 物理ダメージ計算機

ドラゴンクエストウォークの物理ダメージを計算するブラウザ完結型のツールです。  
HTMLファイル単体で動作し、サーバー不要・インストール不要です。

---

## 使い方

`index.html` をブラウザで開くだけで使用できます。  
値を入力するたびに自動で再計算されます。

---

## 入力項目

### 攻撃力・基礎値

| 項目 | 説明 |
|------|------|
| 攻撃力 | キャラクターの攻撃力 |
| バイシオン（段数） | 0〜3段。攻撃力 × 段数 × 20% を加算 |
| レベルアップ（段数） | 0〜20段。`floor(1.04^段数 × 100) / 100` を係数とし、`攻撃力 × (係数 − 1)` を加算 |
| 剣魔合一（加算値） | 攻撃力への直接加算値 |
| 敵の守備力 | 基礎値計算で使用 |

**基礎値** = `floor( (攻撃力 + バイシオン + LvUP + 剣魔合一) / 2 − 守備力 / 4 )`

---

### スキル

| 項目 | 説明 |
|------|------|
| スキル倍率（%） | 使用スキルの倍率（例：覇王斬 = 900） |
| スキル倍率補正（%） | 心珠などによるスキル倍率アップ（例：30×2段 = 60） |

スキル倍率と補正は**別々のステップで乗算・切り捨て**されます。

---

### テンション

| 項目 | 説明 |
|------|------|
| 味方テンション | テンション段階を選択（0 / 5 / 20 / 50 / 100） |

テンション倍率 T = `1 + 味方テンション% / 100`

テンションの影響を受ける項目は以下のとおりで、いずれも **T 乗算後に整数切り捨て**されます：

- 斬体倍率%、属性倍率%、系統倍率%、名称倍率%、会心ダメアップ%
- 斬体耐性%、属性耐性%（敵テンション補正と組み合わせて切り捨て）

---

### 斬体ダメージ

| 項目 | 説明 |
|------|------|
| 斬体倍率（%） | テンション補正あり → `floor(入力値 × T)` |
| スキルダメージアップ（%） | テンション補正なし |
| おうえん（%） | テンション補正なし |

**斬体合計** = `floor( (100 + 斬体% + スキルUP + おうえん) / 100 × 1000 ) / 1000`（小数第3位切り捨て）

---

### 属性ダメージ

| 項目 | 説明 |
|------|------|
| 属性倍率（%） | テンション補正あり → `floor(入力値 × T)` |
| フォースチャージ（段数） | 0〜6段。1段につき +30% |
| フォースブラスト（%） | テンション補正なし |

**属性合計** = `floor( (100 + 属性% + FC×30 + FB) / 100 × 1000 ) / 1000`（小数第3位切り捨て）

---

### 系統・名称ダメージ

| 項目 | 説明 |
|------|------|
| 系統倍率（%） | テンション補正あり → `floor(入力値 × T)` |
| 名称倍率（%） | テンション補正あり → `floor(入力値 × T)` |

**系統合計** = `floor( (floor(系統% × T) + 100) / 100 × 1000 ) / 1000`（小数第3位切り捨て）  
名称合計も同様。

---

### 敵テンション・耐性補正

| 項目 | 説明 |
|------|------|
| 敵テンション | −100〜+100。斬体・属性耐性%の補正に影響 |
| 斬体耐性（%） | `effective% = floor(入力値 × 敵T補正 / 100)` → 倍率 = `(100 − effective%) / 100` |
| 属性耐性（%） | 同上 |
| 因果（段数） | 0〜6段。`1 + 段数 × 20 / 100` |
| フォースブレイク（段数） | 0〜2段。`1 + 段数 × 30 / 100` |
| 閃き | なし（×1.0） / 1段階（×1.4） / 2段階（×1.8）。**会心種別が「通常」の場合は強制 ×1.0** |
| きあいため | なし（×1.0） / きあいため（×2.0） / 超きあいため（×2.25） |

---

### 会心ダメージ

| 項目 | 説明 |
|------|------|
| 会心ダメアップ（%） | テンション補正あり → `floor(入力値 × T)`。会心種別%に**加算**（乗算ではない）。**会心種別が「通常」の場合は無効** |
| 会心ダメージ種別 | 通常（100%） / 会心（160%） / 多段・全体（160〜200%、中央180%） / 超会心（230〜270%、中央250%） |

**会心合計** = `(会心種別% + floor(会心ダメアップ% × T)) / 100`

> 例：超会心250% ＋ 会心ダメアップ13% → `(250 + 13) / 100` = **2.63倍**（乗算ではなく加算）

**会心種別が「通常 100%」の場合：会心ダメアップ・閃きはともに強制 ×1.0 で無視されます。**

---

### 乱数幅

```
乱数幅     = floor( 基礎値 / 16 )
下限基礎値 = floor( 基礎値 − 乱数幅 ) − 1
上限基礎値 = floor( 基礎値 + 乱数幅 ) + 1
```

---

## 計算式（全体フロー）

```
① LvUP係数   = floor( 1.04^段数 × 100 ) / 100        ← 小数第3位切り捨て
   LvUP加算値 = 攻撃力 × ( LvUP係数 − 1 )

② 基礎値     = floor( (攻撃力 + バイシオン + LvUP + 剣魔合一) / 2 − 守備力 / 4 )

③ 乱数幅     = floor( 基礎値 / 16 )
   下限基礎値 = floor( 基礎値 − 乱数幅 ) − 1
   上限基礎値 = floor( 基礎値 + 乱数幅 ) + 1

④ テンション補正（T = 1 + 味方テンション% / 100）
   斬体% / 属性% / 系統% / 名称% / 会心ダメアップ%
     → floor( 入力値 × T )                            ← 整数切り捨て
   斬体耐性% / 属性耐性%
     → floor( 入力値 × 敵T補正 / 100 )                ← 整数切り捨て

⑤ 各倍率（%→倍率変換後は小数第3位切り捨て）
   斬体合計 = floor( (100 + 斬体% + スキルUP + おうえん) / 100 × 1000 ) / 1000
   属性合計 = floor( (100 + 属性% + FC×30 + FB) / 100 × 1000 ) / 1000
   系統合計 = floor( (100 + 系統%) / 100 × 1000 ) / 1000
   名称合計 = floor( (100 + 名称%) / 100 × 1000 ) / 1000
   会心合計 = (会心種別% + 会心ダメアップ%) / 100       ← 加算。通常時は強制 1.0
   閃き     = 選択値                                   ← 通常時は強制 1.0

⑥ 第1段階 = floor( 基礎値 × スキル倍率
                  × 斬体合計 × 属性合計
                  × 系統合計 × 名称合計
                  × 因果 × フォースブレイク
                  × 斬体耐性 × 属性耐性
                  × 閃き × 会心合計 )

⑦ 最終ダメージ = floor( ⑥ × スキル倍率補正 × きあいため )

※ 最小は下限基礎値・会心下限%、最大は上限基礎値・会心上限%で⑥⑦を同様に計算
```

## 計算結果の保存

画面下部の履歴エリアに最大3件まで保存できます。  
各スロットに任意のメモを付けて保存・呼出・消去が可能です（セッション中のみ保持）。

---

## 動作環境

- モダンブラウザ（Chrome / Safari / Firefox / Edge）
- スマートフォン対応（最大幅520pxのモバイルファースト設計）
- サーバー・インターネット接続不要
