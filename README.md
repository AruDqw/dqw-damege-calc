# ⚔️ DQW 物理ダメージ計算機

ドラゴンクエストウォークの物理ダメージを計算するブラウザ完結型のツールです。  
HTMLファイル単体で動作し、サーバー不要・インストール不要です。

---

## 使い方

`index.html` をブラウザで開くだけで使用できます。  
値を入力するたびに自動で再計算されます。

---

## 入力項目

### 攻撃力・基礎値

| 項目 | 説明 |
|------|------|
| 攻撃力 | キャラクターの攻撃力 |
| バイシオン（段数） | 0〜3段。攻撃力 × 段数 × 20% を加算 |
| レベルアップ（段数） | 0〜20段。`floor(1.04^段数 × 100) / 100` で倍率を求め、`floor(攻撃力 × 倍率) − 攻撃力` を加算 |
| 剣魔合一（加算値） | LvUPと同じ倍率で `floor(入力値 × 倍率)` を計算し加算 |
| 敵の守備力 | 基礎値計算で使用 |
| 敵の守備力変化（段数） | −3〜+3。レベルダウン前の守備力 × ±20% × 段数（切り捨て） |
| 敵のレベルダウン（段数） | 0〜10段。`ceil(0.96^段数 × 100) / 100` で係数を算出（1段階=96%、2段階=93%、10段階=67%） |
| 直撃 | あり／なし。ありの場合は最終守備力が 0 になる |

**最終守備力** = `floor(守備力 × LvDown係数) + 守備力変化`（直撃時は 0）

**基礎値** = `floor( (攻撃力 + バイシオン + LvUP + 剣魔合一) / 2 − 最終守備力 / 4 )`

**乱数幅** = `floor( 基礎値 / 16 )`  
**下限基礎値** = `floor( 基礎値 − 乱数幅 ) − 1`  
**上限基礎値** = `floor( 基礎値 + 乱数幅 ) + 1`

---

### スキル

| 項目 | 説明 |
|------|------|
| スキル倍率（%） | 使用スキルの倍率（例：覇王斬 = 900）。**第1段階**で乗算 |
| スキル倍率補正（%） | 心珠などによるスキル倍率アップ（例：30×2段 = 60）。**第2段階**で乗算 |

スキル倍率と補正は**別々のステップで乗算・切り捨て**されます。

---

### テンション

| 項目 | 説明 |
|------|------|
| 味方テンション | テンション段階を選択（0 / 5 / 20 / 50 / 100） |

テンション倍率 T = `1 + 味方テンション% / 100`

テンションの影響を受ける項目は以下のとおりで、いずれも **T 乗算後に整数切り捨て**されます：

- 斬体倍率%、属性倍率%、系統倍率%、名称倍率%、会心ダメアップ%
- 斬体耐性%、属性耐性%（敵テンション補正と組み合わせて切り捨て）

---

### 斬体ダメージ

| 項目 | 説明 |
|------|------|
| 斬体倍率（%） | テンション補正あり → `floor(入力値 × T)` |
| スキルダメージアップ（段数） | 1段階 +20%。ラーミアなど |
| 斬体ダメージアップ（段数） | 1段階 +20%。円月輪など |
| おうえん | あり +80% / なし |

**斬体合計** = `floor( (100 + 斬体% + スキルUP + 斬体UP + おうえん) / 100 × 1000 ) / 1000`（小数第3位切り捨て）

---

### 属性ダメージ

| 項目 | 説明 |
|------|------|
| 属性倍率（%） | テンション補正あり → `floor(入力値 × T)` |
| フォースチャージ（段数） | 0〜6段。1段につき +30% |
| フォースブラスト（%） | テンション補正なし |

**属性合計** = `floor( (100 + 属性% + FC×30 + FB) / 100 × 1000 ) / 1000`（小数第3位切り捨て）

---

### 系統・名称ダメージ

| 項目 | 説明 |
|------|------|
| 系統倍率（%） | テンション補正あり → `floor(入力値 × T)` |
| 名称倍率（%） | テンション補正あり → `floor(入力値 × T)` |

**系統合計** = `floor( (floor(系統% × T) + 100) / 100 × 1000 ) / 1000`（小数第3位切り捨て）  
名称合計も同様。

---

### 敵テンション・耐性補正

| 項目 | 説明 |
|------|------|
| 敵テンション | −100〜+100。斬体・属性耐性%の補正に影響 |
| 斬体耐性（%） | `effective% = floor(入力値 × 敵T補正 / 100)` → 倍率 = `(100 − effective%) / 100` |
| 属性耐性（%） | 同上 |
| 因果（段数） | 0〜6段。`1 + 段数 × 20 / 100` |
| フォースブレイク（段数） | 0〜2段。フォースブレイクと属性耐性変化を合算して倍率を算出 |
| 属性耐性変化（段数） | −3〜+3。1段階で属性耐性 −20%。フォースブレイクと加算 |
| 閃き | なし（×1.0） / 1段階（×1.4） / 2段階（×1.8）。**会心種別が「通常」の場合は強制 ×1.0** |
| きあいため | なし（×1.0） / きあいため（×2.0） / 超きあいため（×2.25）。**第2段階**で乗算 |

**フォースブレイク＋属性耐性変化合計** = `1 + (フォースブレイク段数 × 30 − 属性耐性変化段数 × 20) / 100`

> 例：フォースブレイク2段階＋属性耐性変化+1段階 → `1 + (60 − 20) / 100` = **1.400**

---

### 斬体耐性変化

| 項目 | 説明 |
|------|------|
| 斬体耐性変化 | −3〜+3段階（棺など） |
| 衰弱 | 0〜3段階。1段階 +10%（ラーミアなど） |
| 凍結 | 0〜2段階。1段階 +10%（グリ爪など） |
| 戦意喪失 | 0〜3段階。1段階 +5% |

**斬体耐性変化合計** = `(100 + 斬体耐性変化 + 衰弱 + 凍結 + 戦意喪失) / 100`

---

### 会心ダメージ

| 項目 | 説明 |
|------|------|
| 会心ダメアップ（%） | テンション補正あり → `floor(入力値 × T)`。会心種別%に**加算**（乗算ではない）。**会心種別が「通常」の場合は無効** |
| 会心ダメージ種別 | 通常（100%） / 会心（160%） / 多段・全体（160〜200%、中央180%） / 超会心（230〜270%、中央250%） |

**会心合計** = `(会心種別% + floor(会心ダメアップ% × T)) / 100`

> 例：超会心250% ＋ 会心ダメアップ13% → `(250 + 13) / 100` = **2.63倍**（乗算ではなく加算）

**会心種別が「通常 100%」の場合：会心ダメアップ・閃きはともに強制 ×1.0 で無視されます。**

---

## 計算式（全体フロー）

```
① LvUP倍率   = floor( 1.04^段数 × 100 ) / 100        ← 倍率を先に小数第2位切り捨て
   LvUP加算値 = floor( 攻撃力 × LvUP倍率 ) − 攻撃力
   剣魔合一   = floor( 入力値 × LvUP倍率 )             ← 同じ倍率を使用

② 敵LvDown係数 = ceil( 0.96^段数 × 100 ) / 100        ← 切り上げ（lv1=96%、lv2=93%、lv10=67%）
   最終守備力  = floor( 守備力 × LvDown係数 ) + 守備力変化　（直撃時は 0）

③ 基礎値     = floor( (攻撃力 + バイシオン + LvUP + 剣魔合一) / 2 − 最終守備力 / 4 )

④ 乱数幅     = floor( 基礎値 / 16 )
   下限基礎値 = floor( 基礎値 − 乱数幅 ) − 1
   上限基礎値 = floor( 基礎値 + 乱数幅 ) + 1

⑤ テンション補正（T = 1 + 味方テンション% / 100）
   斬体% / 属性% / 系統% / 名称% / 会心ダメアップ%
     → floor( 入力値 × T )                            ← 整数切り捨て
   斬体耐性% / 属性耐性%
     → floor( 入力値 × 敵T補正 / 100 )                ← 整数切り捨て

⑥ 各倍率（%→倍率変換後は小数第3位切り捨て）
   斬体合計       = floor( (100 + 斬体% + スキルUP + 斬体UP + おうえん) / 100 × 1000 ) / 1000
   属性合計       = floor( (100 + 属性% + FC×30 + FB) / 100 × 1000 ) / 1000
   系統合計       = floor( (100 + 系統%) / 100 × 1000 ) / 1000
   名称合計       = floor( (100 + 名称%) / 100 × 1000 ) / 1000
   FB＋属性耐性変化 = 1 + (FB段数×30 − 属性耐性変化段数×20) / 100
   会心合計       = (会心種別% + 会心ダメアップ%) / 100  ← 加算。通常時は強制 1.0
   閃き           = 選択値                              ← 通常時は強制 1.0
   斬体耐性変化合計 = (100 + 斬体耐性変化 + 衰弱 + 凍結 + 戦意喪失) / 100

⑦【第1段階】= floor( 基礎値 × スキル倍率
                    × 斬体合計 × 属性合計
                    × 系統合計 × 名称合計
                    × 因果 × (FB＋属性耐性変化)
                    × 斬体耐性 × 属性耐性
                    × 閃き × 会心合計
                    × 斬体耐性変化合計 )

⑧【第2段階】最終ダメージ = floor( ⑦ × スキル倍率補正 × きあいため )

※ 最小は下限基礎値・会心下限%、最大は上限基礎値・会心上限%で⑦⑧を同様に計算
```

---

## 色分けについて

各項目の右側の結果欄は計算ステップごとに色分けされています。

- 🟢 **緑**：第1段階（`floor(…)` の一括乗算に使われる要素）
- 🟠 **橙**：第2段階（スキル倍率補正・きあいため）
- ⬜ **グレー**：参考値・中間値（乗算には直接使われない）

---

## 計算結果の保存

画面下部の履歴エリアに最大3件まで保存できます。  
各スロットに任意のメモを付けて保存・呼出・消去が可能です（セッション中のみ保持）。

## 動作環境

- モダンブラウザ（Chrome / Safari / Firefox / Edge）
- スマートフォン対応（最大幅520pxのモバイルファースト設計）
- サーバー・インターネット接続不要
