計算式：
① LvUP倍率：floor(1.04^段数 × 100) / 100　→　LvUP加算値 = floor(攻撃力 × 倍率) − 攻撃力
 剣魔合一：floor(入力値 × 同倍率)　← LvUPと同じ倍率を使用
② 敵レベルダウン：ceil(0.96^段数 × 100)/100 で係数算出（lv1=96%、lv2=93%、lv10=67%）
 敵守備力変化：レベルダウン前の守備力 × ±20%×段数（切り捨て）
 最終守備力 = floor(守備力 × LvDown係数) + 守備力変化
③ 基礎値 = floor( (攻撃力 + バイシオン + LvUP + 剣魔合一) / 2 − 最終守備力 / 4 )
 乱数幅 = floor( 基礎値 / 16 )　→　下限 = floor(基礎値 − 乱数幅) − 1　／　上限 = floor(基礎値 + 乱数幅) + 1
④ テンション補正（T = 1 + 味方テンション% / 100）は以下の%値に影響：
 ・斬体% / 属性% / 系統% / 名称% / 会心ダメアップ% → floor( 入力値 × T )　← 整数切り捨て
 ・斬体耐性% / 属性耐性% → floor( 入力値 × 敵T補正 / 100 )　← 整数切り捨て
⑤ 各倍率（%→倍率変換後は小数第3位切り捨て）：
 斬体合計 = floor( (100 + 斬体% + スキルUP + 斬体UP + おうえん) / 100 × 1000 ) / 1000
 属性合計 = floor( (100 + 属性% + FC×30 + FB) / 100 × 1000 ) / 1000
 系統合計 = floor( (100 + 系統%) / 100 × 1000 ) / 1000　／　名称合計 = floor( (100 + 名称%) / 100 × 1000 ) / 1000
 会心合計 = (会心種別% + 会心ダメアップ%) / 100　← 会心ダメアップは加算
 斬体耐性合計 = (100 + 斬体耐性変化 + 衰弱 + 凍結 + 戦意喪失) / 100
⑥【第1段階】= floor( 基礎値 × スキル倍率 × 斬体合計 × 属性合計 × 系統合計 × 名称合計 × 因果(1+段数×20%+極み30%) × (フォースブレイク＋属性耐性変化) × 斬体耐性 × 属性耐性 × 閃き × 会心合計 × 斬体耐性合計 )
⑦【第2段階】最終ダメージ = floor( ⑥ × スキル倍率補正 × きあいため )
※ 最小は下限基礎値・会心下限%、最大は上限基礎値・会心上限%で同様に計算
